### Creating a Task

1. Navigate to the **Tasks** tab
2. Tap the **"+" button** (usually bottom-right)
3. Fill in task details:
   - **Title** (required): Brief description
   - **Description** (optional): Detailed information
   - **Priority**: Choose High, Medium, or Low
   - **Category**: Design, Dev, Marketing, Research, or Bug
   - **Due Date**: Set a deadline
   - **Assignees**: Assign to team members
   - **Team**: Associate with a team
4. Tap **"Create"** or **"Save"**

### Working with Subtasks

Break large tasks into smaller pieces:

1. Open task details
2. Find the **Subtasks** section
3. Tap **"Add Subtask"**
4. Enter subtask title
5. Repeat for more subtasks

**Track Progress**:
- Check off subtasks as you complete them
- View progress bar showing completion percentage
- See "X of Y completed" counter

### Due Dates and Reminders

**Set Due Dates**:
1. Edit task
2. Tap the **Due Date** field
3. Select date from calendar
4. Save task

**Overdue Tasks**:
- Highlighted in red
- Marked with "Overdue" badge
- Sorted to top in filtered views

**Reminders**:
- Toggle "Remind Me" option
- Get notifications before due date

### Deleting a Task

1. Open task details
2. Tap the **Delete** button (trash icon)
3. Confirm deletion
4. Task is removed permanently

## â“ Common Questions

### How do I mark a task as complete?

Tap the task, then change status to "Completed" or use the quick complete button.

### Can I assign a task to multiple people?

Yes! When creating/editing a task, you can select multiple assignees.

### How do I know if a task is overdue?

Overdue tasks are highlighted in red and show an "Overdue" badge.

### Can I work offline?

Yes! Task Flow works fully offline. Changes sync when you're back online (sync coming soon).

### How do I delete a team?

Open team details and tap the delete button. Note: This removes all team tasks.

### Can I customize task statuses?

Yes! Each team can define custom task statuses in team settings.
