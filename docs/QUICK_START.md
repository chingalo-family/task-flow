## ğŸ“ First Steps (3 minutes)

### 1. Create Your Account (30 seconds)

1. Open Task Flow
2. Tap "Sign Up"
3. Enter:
   - Username
   - Email
   - Password
4. Tap "Create Account"

### 2. Create Your First Task (30 seconds)

1. Skip or complete the onboarding
2. Tap the **Tasks** tab (bottom navigation)
3. Tap the **"+" button** (bottom-right)
4. Enter:
   - **Title**: "My First Task"
   - **Priority**: High (optional)
   - **Due Date**: Tomorrow (optional)
5. Tap **"Create"**

ğŸ‰ **You created your first task!**

### 3. Create a Team (1 minute)

1. Tap the **Teams** tab
2. Tap the **"+" button**
3. Enter:
   - **Name**: "My Team"
   - **Description**: "Getting started with Task Flow"
   - Choose an **icon** and **color**
4. Tap **"Create Team"**

### 4. Create a Team Task (30 seconds)

1. Go back to **Tasks** tab
2. Tap **"+" button**
3. Enter task title
4. Select your team from **Team** dropdown
5. Tap **"Create"**

### 5. Check Notifications (30 seconds)

1. Tap the **Notifications** tab
2. See your recent activity
3. Try the filter chips (All, Unread, etc.)
